## 1. 系统概述

### 1.1 项目定位

AstroAI-Core 是一个开源的天文数据分析平台，利用深度学习技术帮助用户：
- 自动分类恒星/星系
- 探测系外行星候选体
- 识别异常天体
- 分析光谱数据

### 1.2 核心功能

| 功能模块 | 描述 | 优先级 |
|----------|------|--------|
| 光变曲线分析 | 处理 TESS/Kepler 时序数据，探测凌星信号 | P0 |
| 图像分类 | 恒星/星系/星云自动分类 | P0 |
| 异常检测 | 发现罕见/未知天体类型 | P1 |
| 光谱分析 | 恒星光谱分类、红移测量 | P1 |
| 数据可视化 | 交互式图表、3D 星图 | P1 |
| API 服务 | RESTful API 供第三方调用 | P2 |

### 3.2 核心模型详情

#### 3.2.1 ExoPlanetNet (系外行星探测)

**输入：** 光变曲线时序数据 (TESS/Kepler)  
**输出：** 凌星信号概率 + 行星参数估计

```python
class ExoPlanetNet(nn.Module):
    def __init__(self):
        super().__init__()
        # 1D CNN 提取局部特征
        self.conv1d = nn.Sequential(
            nn.Conv1d(1, 64, kernel_size=7, padding=3),
            nn.BatchNorm1d(64),
            nn.ReLU(),
            nn.Conv1d(64, 128, kernel_size=5, padding=2),
            nn.BatchNorm1d(128),
            nn.ReLU(),
        )
        # LSTM 捕捉时序依赖
        self.lstm = nn.LSTM(128, 256, num_layers=2, 
                           bidirectional=True, dropout=0.3)
        # 注意力机制
        self.attention = nn.MultiheadAttention(256, 8)
        # 分类头
        self.classifier = nn.Sequential(
            nn.Linear(512, 256),
            nn.ReLU(),
            nn.Dropout(0.5),
            nn.Linear(256, 2)  # 行星/非行星
        )
    
    def forward(self, x):
        # x: [batch, seq_len, 1]
        x = x.transpose(1, 2)  # [batch, 1, seq_len]
        x = self.conv1d(x)     # [batch, 128, seq_len]
        x = x.transpose(0, 2)  # [seq_len, batch, 128]
        x, _ = self.lstm(x)    # [seq_len, batch, 512]
        # 注意力 pooling
        x, _ = self.attention(x, x, x)
        x = x.mean(dim=0)      # [batch, 512]
        return self.classifier(x)
```

**训练数据：**
- TESS Object of Interest (TOI) 目录：~8000 候选体
- Kepler 确认行星：~2700 颗
- 负样本：随机恒星光变曲线

**预期性能：**
- 准确率：> 90%
- 召回率：> 85%
- 推理时间：< 100ms/样本

#### 3.2.2 StarClassifier (恒星分类)

**输入：** 恒星图像 (多波段) 或光谱  
**输出：** 光谱类型 (OBAFGKM) + 光度级

**架构：** ResNet50 + Vision Transformer 混合

#### 3.2.3 AnomalyDetector (异常检测)

**输入：** 任意天文数据 (图像/光谱/时序)  
**输出：** 异常分数 + 异常类型

**架构：** Variational Autoencoder  
**训练方式：** 无监督，仅使用正常样本

### 4.2 数据预处理

```python
# 光变曲线预处理示例
def preprocess_lightcurve(raw_data: np.ndarray) -> np.ndarray:
    """
    光变曲线预处理流程
    """
    # 1. 去趋势 (detrending)
    detrended = remove_systematic_trend(raw_data)
    
    # 2. 归一化
    normalized = (detrended - np.mean(detrended)) / np.std(detrended)
    
    # 3. 缺失值插值
    filled = interpolate_gaps(normalized, method='spline')
    
    # 4. 重采样到统一长度
    resampled = resample(filled, target_length=1024)
    
    # 5. 添加噪声增强 (训练时)
    if training:
        resampled = add_gaussian_noise(resampled, sigma=0.01)
    
    return resampled
```

### 4.3 特征工程

| 数据类型 | 特征 | 提取方法 |
|----------|------|----------|
| 光变曲线 | 周期、深度、持续时间 | 折叠 + 拟合 |
| 图像 | 形态参数、颜色指数 | SExtractor |
| 光谱 | 吸收线等值宽度、红移 | 谱线拟合 |
| 星表 | 位置、自行、视差 | Gaia DR3 |

## 5. API 设计

### 5.1 RESTful API

```yaml
openapi: 3.0.0
info:
  title: AstroAI-Core API
  version: 1.0.0

paths:
  /api/v1/analyze/lightcurve:
    post:
      summary: 分析光变曲线
      requestBody:
        content:
          application/json:
            schema:
              type: object
              properties:
                data: { type: array, items: { type: number } }
                cadence: { type: number }
      responses:
        200:
          description: 分析结果
          content:
            application/json:
              schema:
                type: object
                properties:
                  planet_probability: { type: number }
                  period: { type: number }
                  depth: { type: number }

  /api/v1/classify/image:
    post:
      summary: 图像分类
      requestBody:
        content:
          multipart/form-data:
            schema:
              type: object
              properties:
                image: { type: string, format: binary }
      responses:
        200:
          description: 分类结果

  /api/v1/detect/anomaly:
    post:
      summary: 异常检测
      # ...

  /api/v1/objects/{id}:
    get:
      summary: 获取天体信息
      parameters:
        - name: id
          in: path
          required: true
          schema: { type: string }
```

### 5.2 Python SDK

```python
from astroai import AstroAIClient

client = AstroAIClient(api_key="your_key")
```
