## 1. 商业模式概述

采用 **Freemium + B2B** 混合模式：
- **免费层**：吸引用户、建立社区
- **付费层**：高级功能、专业用户
- **B2B 服务**：教育机构、研究团队、商业客户

```
┌─────────────────────────────────────────────────────────────┐
│                    收入结构                                 │
│                                                             │
│   B2C 订阅 (60%)         B2B 服务 (30%)       其他 (10%)   │
│   ┌─────────────┐       ┌─────────────┐     ┌───────────┐  │
│   │ Pro 订阅    │       │ 教育机构    │     │ 捐赠      │  │
│   │ Premium 订阅│       │ 研究团队    │     │ 周边商品  │  │
│   │ 企业订阅    │       │ 定制开发    │     │ API 超额  │  │
│   └─────────────┘       └─────────────┘     └───────────┘  │
└─────────────────────────────────────────────────────────────┘
```

## 3. 定价策略

### 3.1 订阅定价

| 计划 | 月付 | 年付 (省 20%) | 目标转化率 |
|------|------|---------------|------------|
| Free | $0 | $0 | - |
| Pro | $19.99 | $191.90 | 5% |
| Premium | $49.99 | $479.90 | 1% |
| Enterprise | 定制 | 定制 | 0.1% |

**定价依据：**
- 竞品参考：SkySafari $39.99/年，AstroBin $48/年
- 价值定价：专业用户每月可节省 20+ 小时分析时间
- 心理定价：$19.99 比 $20 感知更低

### 3.2 API 定价

| 层级 | 月配额 | 单价 | 超额价格 |
|------|--------|------|----------|
| Free | 100 次 | $0 | - |
| Developer | 10,000 次 | $29/月 | $0.005/次 |
| Business | 100,000 次 | $199/月 | $0.003/次 |
| Enterprise | 1,000,000+ 次 | 定制 | 定制 |

### 3.3 教育优惠

- 学校/大学：**50% 折扣**
- 学生个人：**免费 Premium** (凭学生证)
- 研究机构：**免费 Enterprise** (需论文署名)

## 6. 变现优化策略

### 6.1 转化漏斗优化

```
访客 (100%)
  │
  │  Landing Page 优化
  ▼
注册用户 (40%)
  │
  │  新手引导 + 首次分析成功
  ▼
活跃用户 (60% of 注册 = 24%)
  │
  │  功能限制提示 + 价值展示
  ▼
付费用户 (8% of 活跃 = 1.9% of 访客)
  │
  │  续费提醒 + 升级推荐
  ▼
留存用户 (70% of 付费)
```

**优化措施：**
1. **A/B 测试**：定价页、CTA 按钮、邮件文案
2. **个性化推荐**：基于使用行为推荐合适套餐
3. **限时优惠**：新用户首月 50% OFF
4. **推荐奖励**：邀请好友得 1 个月免费

### 6.2 用户留存策略

| 策略 | 实施方式 | 预期效果 |
|------|----------|----------|
| 成就系统 | 分析里程碑徽章 | +15% 留存 |
| 定期报告 | 月度分析总结邮件 | +10% 活跃 |
| 新功能通知 | 产品更新推送 | +5% 回访 |
| 社区活动 | 月度竞赛/挑战 | +20% 粘性 |

### 6.3 扩展收入来源

**Phase 2+ 变现扩展：**

1. **数据市场**
   - 用户可出售标注数据

3. **出版合作**
   - 与天文杂志合作专栏

4. **硬件集成**
   - 与望远镜厂商合作
   - 预装 AstroAI 软件

## 7. 风险与应对

| 风险 | 影响 | 应对措施 |
|------|------|----------|
| 用户增长缓慢 | 高 | 加大营销投入、优化转化漏斗 |
| 竞品降价 | 中 | 强调差异化、增加功能壁垒 |
| API 成本超支 | 中 | 优化模型、引入缓存层 |
| 付费转化低 | 高 | 调整定价、增加免费层限制 |
| 政策变化 | 低 | 多元化数据源、合规审查 |

## 8. 关键指标 (KPIs)

### 8.1 增长指标
- 月新增注册用户
- 用户获取成本 (CAC)
- 病毒系数 (K-factor)

### 8.2 收入指标
- 月经常性收入 (MRR)
- 平均每用户收入 (ARPU)
- 客户生命周期价值 (LTV)

### 8.3 留存指标
- 日/月活跃用户 (DAU/MAU)
- 付费用户留存率
- 流失率 (Churn Rate)

### 8.4 产品指标
- 分析任务完成数
- 平均会话时长
- 功能使用率
