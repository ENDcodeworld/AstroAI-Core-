# 🚀 AstroAI-Core Frontend - Quick Start

## One-Command Setup

```bash
cd /home/admin/.openclaw/workspace/projects/AstroAI-Core/frontend && npm install && npm run dev
```

## What You Get

✅ **Exoplanet Browser** - Browse and filter discovered exoplanets  
✅ **3D Orbit Viewer** - Interactive orbital visualizations  
✅ **AI Image Classifier** - Upload and classify astronomical images  
✅ **Data Visualization** - Charts and statistics dashboard  
✅ **Responsive Design** - Works on mobile, tablet, and desktop  

## Access the App

Once running, open: **http://localhost:3000**

## Navigation

- **Exoplanets** → Browse planet catalog
- **Image Classifier** → Upload images for AI classification
- **Visualization** → View data charts and statistics

## Demo Features

### Exoplanet Browser
1. Use search to find planets by name
2. Filter by constellation or discovery method
3. Toggle "Habitable Only" filter
4. Click on any planet card to see details
5. View 3D orbital visualization

### Image Classifier
1. Drag and drop images or click to upload
2. View classification results with confidence scores
3. Switch to History tab to see past classifications
4. Try uploading multiple images at once

### Data Visualization
1. Toggle between Chart, Grid, and Stats views
2. Filter by constellation
3. Explore interactive charts
4. View distribution statistics

## Development Commands

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Run linter
npm run lint
```

## Tech Stack

- ⚛️ React 18 + TypeScript
- 🎨 Material-UI (Dark Theme)
- 🌌 Three.js for 3D visualizations
- 📊 Chart.js for data visualization
- 🐻 Zustand for state management
- ⚡ Vite for fast builds
