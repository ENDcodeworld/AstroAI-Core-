# AstroAI Web Frontend

React + TypeScript frontend for AstroAI-Core.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Three.js (3D visualization)
- React Query (data fetching)
- Zustand (state management)
